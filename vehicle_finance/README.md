# 🚗 車両維持費管理システム (Vehicle Finance Calculator)

車の維持費を記録・分析するアプリケーションです。

## 📋 機能

### 現在実装済み
- ✅ **車両登録**: 複数の車両を登録・管理
- ✅ **実績入力**: 6種類の維持費を記録
  - ⛽ ガソリン代
  - 🔧 車検費用
  - 🛡️ 保険料（任意保険）
  - 🏷️ 自動車税
  - 🅿️ 駐車場代
  - 🔧 メンテナンス費用
- ✅ **ダッシュボード**: 維持費の集計と可視化
- ✅ **実績一覧**: カテゴリ別の詳細表示
- ✅ **データ保存**: 自動保存・読み込み機能

### 今後実装予定
- 📊 複数車種の比較分析
- 🔍 期間別の費用推移分析
- 📈 将来コストのシミュレーション
- 💾 Excel/CSVエクスポート

## 🚀 起動方法

### 1. 仮想環境のアクティベート
```powershell
.\.venv\Scripts\Activate.ps1
```

### 2. アプリの起動
```powershell
python run_vehicle_app.py
```

または直接Streamlitコマンドで：
```powershell
streamlit run vehicle_finance/ui/streamlit_app.py --server.port=8509
```

### 3. ブラウザでアクセス
アプリが起動すると、自動的にブラウザで開きます。
手動でアクセスする場合: http://localhost:8509

## 💡 使い方

### 1. 車両を登録
サイドバーの「➕ 新規車両登録」から車両情報を入力

### 2. 実績を入力
「➕ 実績入力」タブから各種維持費を記録
- ガソリン代: 給油日、給油量、単価
- 車検費用: 車検基本料、重量税、自賠責保険など
- 保険料: 年払い・月払いの保険料
- 自動車税: 自動車税・軽自動車税
- 駐車場代: 月額・年額の駐車料金
- メンテナンス: オイル交換、タイヤ交換、修理など

### 3. データを確認
- **ダッシュボード**: 集計データと円グラフ
- **実績一覧**: カテゴリ別の詳細表示

## 📊 データ保存

データは `vehicle_finance/data/saved_data/expense_data.pkl` に自動保存されます。
アプリを再起動しても、データは保持されます。

## 🐛 トラブルシューティング

### データが表示されない
- サイドバーから車両を選択してください
- 車両が未登録の場合は、まず車両を登録してください

### データが保存されない
- `vehicle_finance/data/saved_data/` ディレクトリへの書き込み権限を確認してください

### ポート8509が使用中
起動スクリプトのポート番号を変更してください：
```python
"--server.port=8510",  # 別のポートに変更
```
